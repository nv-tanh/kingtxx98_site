Requirement Design Implement

1. Chọn một đối tượng bất kì trong thế giới thực, ví dụ như sinh viên, môn học, giảng viên,... để thiết kế trên phần mềm một chức năng trên web để thêm, xóa,sửa, tìm kiếm cho đối tượng đó.

2. Upload tất cả lên Azure / Github trong menu Lab1 theo thứ tự:
    + Yêu cầu: copy lại nội dung của phần 1 vào đây (trình bày cho đẹp).
    + Thiết kế: để link tới các file sau đây:
        - Thiết kế trên giấy (chụp lại ảnh).
        -Thiết kế trên phần mềm thiết kế (Figma).
     + Hiện thực: chuyển tất cả các thiết kế ở trên sang file HTML.
