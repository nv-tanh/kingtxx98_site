Requirement User Scenario Design Implement

1. Tạo ra một user scenario.

2. Chọn 1 bước trong user scenario để thiết kế 1 form giao diện Web trên công cụ thiết kế đã chọn. Sau đó tạo ra file html tương ứng cho thiết kế này.

User scenario Booking hotel online user scenario

    1. Truy cập vào địa chỉ trang đặt vé online.
    2. Nhập thông tin vào form có sẵn (địa điểm, thời gian đến, thời gian đi, số lượng người) và ấn nút tìm kiếm.
    3. Lựa chọn khách sạn phù hợp trong các khách tìm được để xem chi tiết về khách sạn.
    4. Lựa chọn các phòng phù hợp và tiến hành thanh toán.
    5. Điền thông tin vào đơn đặt phòng.
    6. Kết thúc.
